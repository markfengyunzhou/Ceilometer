## 概念介绍

 Project > Resource > Meter
 
* Project: 项目 （身份管理-项目）

* Resource: 资源 （如：虚拟机）

* Meter: 计量项 （如：镜像、disk...）

* Sample: 某个 Poject 的某个 Resource 在某一时刻某 Meter 的数值 （如：cpu利用率、disk IO...）
  
     数值类型：

        * cumulative：累计值

        * delta：变化值

        * gauge：离散或者波动值

* Statistics: 某区间 Samples 的聚合值 （Count、Max、Min、Avg、Sum）
